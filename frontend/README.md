# UniApply Frontend

React-based frontend for the Unified University Application Platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The frontend will run on `http://localhost:3000` and proxy API requests to `http://localhost:5000`.

## Build

To create a production build:
```bash
npm run build
```

## Features

- Student portal with application management
- Admin panel with comprehensive management tools
- Responsive design with Tailwind CSS
- Modern React patterns with hooks and context
